手机加速感应器

Accelerometer 模块管理设备加速度传感器，用于获取设备加速度信息，包括 x （屏幕水平方向）、y（屏幕垂直方向）、z（垂直屏幕平面方向）三个方向的加速度。 由于重力加速度的影响，即使静止不动的情况下，垂直方向也会有加速度。
    应用场景
  • 摇一摇
  • 计步
  • 报警系统
  • 速度感应器
    调用接口
  • 即时获取
      getCurrentAcceleration
    • void plus.accelerometer.getCurrentAcceleration(successCB, errorCB)
  • 监听接口
      watchAcceleration
    • Number plus.accelerometer.watchAcceleration(successCB, errorCB, option)
    • Option:{frequency:1000}
  • 取消监听
      clearWatch
    • void plus.accelerometer.clearWatch(watchId)

手机录音和播放

Audio 模块用于提供音频的录制和播放功能，可以调用系统的麦克风设备进行录音操作，也可以调用系统的扬声器设备播放音频文件。
    应用场景
  • 音频录制
  • 语音聊天
  • 语音留言
  • 音频播放
    调用接口
  • 对象
      录音
    • AudioRecorder
      • void recorder.record(option, successCB, errorCB)
        • option:{filename:"_doc/audio/"}
      播放
    • AudioPlayer
      • void play(successCB, errorCB)
      • void pause()
      • void resume()
      • void stop()
      • seekTo(position)
        • Position:100 单位：秒
      • Number getDuration() 单位：秒
      • Number getPosition() 单位：秒
      • void setRoute(route)
        • Route: //数字。常量 ROUTE_SPEAKER （扬声器模式）； ROUTE_EARPIECE （听筒模式）
  • 方法
      获取录音设备
    • getRecorder()
      • AudioRecorder plus.audio.getRecorder()
      创建播放器
    • createPlayer()
      • AudioPlayer plus.audio.createPlayer(path)

手机扫描二维码

Barcode 模块管理条码扫描，提供常见的条码（二维码即一维码）的扫描识别功能，可调用设备的摄像头对条码图片扫描进行数据输入。
    应用场景
  • 扫码关注
  • 扫码支付
  • 扫码登录
  • 扫码读取
    调用方式
  • 摄像头扫描
  • 图片识别
    条码类型 -- plus.barcode.*
  • QR：二维码 -- 数值为 0
  • EAN13： 条形码（一维条码标准版） -- 数值为 1
  • EAN8： 条形码（一维条码简版） -- 数值为 2
  • ...其他条码类型不常用...
    对象。摄像头扫描
  • Barcode = new plus.barcode.Barcode(id, filters, styles)
    • id： html 对象 id 用户识别控件的初始化
    • filter： 要识别的条码类型过滤器，为条码类型常量数组
    • styles： 条码识别控件样式
      • String frameColor 扫描框颜色
      • String scanbarColor 扫描条颜色
      • String background 条码识别控件背景色
  • Barcode
    • void start(options)
      • options
        • Boolean conserve 是否保存截图。默认 false
        • String filename 截图保存路径
        • Boolean vibrate 成功后是否震动。默认 true
        • String sound 成功后提示音。可为 none。默认 default
    • void cancel()
    • void close()
    • void setFlash(open) 闪光灯
      • open： true/false。默认 false
    • void onmarked() 识别成功后的回调
    • void onerror()
  • 方法。识别图片
    • void plus.barcode.scan(path, successCB, errorCB, filters)
